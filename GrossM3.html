All Embed Gross Description Generator V2

The Command consists of two parts, use "/" as separator

Part I - Either 3-character format #$% or 4-character format #$%&.

# refers to number of pieces (0-9, m/M = multiple, u/U = unspec piece(s), x/X = pieces/piece)

$ refers to number of fragments (0-9, m/M = multiple, u/U = unspec fragment(s), x/X = fragments/fragment)

% refers to number of blocks embedded (0-9)

& is optional, is either omitted, t = tiny, m = mucoid, s = scanty

Part II - Gross dimension part, measurement in mm, accepting characters 0-9 ~ . < >.

If rightmost letter is a (or A), then "xx mm in aggregate." will be used.

Remark: if not following the above format, it will be treated as free text. Between 2 sublabels, use "," as separator.

For example:011/~<0.5, 211/~2, mm1/1-3, m01/~5, mm2m/~15a ,mx1/5a,No tissue seen. For Agar block., **pending**,101/1, 0m1/3a, mm1/<0.5-2
